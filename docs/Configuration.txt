% Configuration

The Google maps API is a free service that is available to everypage that 
is free for its users. You can check the specific terms of service [here.][terms] 

The version 3.0 of the Google Maps API doesn't require a key. In order to use it, 
the page only requires to reference it:

    #xml
    <script type="text/javascript"
        src="http://maps.google.com/maps/api/js?sensor=set_to_true_or_false">

You should replace the sensor variable with either true or false in case you want to 
use the geolocation sensor.

[terms]: http://code.google.com/apis/maps/terms.html
